## Pour commencer

open 2 terminal tabs into the folder.

First tab : 

cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
flask run --port 5000

Second tab :

cd frontend
npm install
npx expo start //add flag --reset-cache if UI issue.
